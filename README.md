# random-str-msg-ts

The random-str-msg-ts package is a TypeScript library that allows you to print a random message to the console. It is designed to be used as a tool for generating random commit messages or any other use case where you need a random string message.

## Install

```npm
npm install -g random-str-msg-ts
```

# Usage

```bash
 random-str-msg-ts
```
